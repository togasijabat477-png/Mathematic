Definisi, bentuk umum, dan contoh dalam kehidupan sehari-hari

Persamaan Linear Satu Variabel (PLSV)

Pengertian, bentuk umum ax + b = 0, dan cara menyelesaikan PLSV

Persamaan Linear Dua Variabel (PLDV)

Bentuk umum ax + by = c, himpunan penyelesaian, dan grafik persamaan linear

SPLDV - Sistem Persamaan Linear Dua Variabel

Pengertian, bentuk umum, dan jenis-jenis penyelesaian (unik, tak hingga, tidak ada)

Metode Grafik SPLDV

Menyelesaikan SPLDV dengan menggambar grafik dan menentukan titik potong

Metode Substitusi SPLDV

Cara menyelesaikan SPLDV dengan metode substitusi (mengganti variabel)

Metode Eliminasi SPLDV

Menyelesaikan SPLDV dengan menghilangkan (mengeliminasi) salah satu variabel

Metode Campuran (Eliminasi dan Substitusi) SPLDV

Kombinasi metode eliminasi dan substitusi untuk menyelesaikan SPLDV

SPLTV - Sistem Persamaan Linear Tiga Variabel

Pengertian SPLTV, bentuk umum ax + by + cz = d, dan contoh soal

Metode Substitusi SPLTV

Menyelesaikan SPLTV dengan metode substitusi bertahap

Metode Eliminasi SPLTV

Menyelesaikan SPLTV dengan eliminasi variabel secara bertahap

Metode Determinan (Aturan Cramer) untuk SPL

Menyelesaikan SPL menggunakan determinan matriks dan aturan Cramer

Metode Matriks Invers untuk SPL

Menyelesaikan SPL dengan bentuk AX = B, maka X = A^(-1) B

Sistem Persamaan Linear Homogen

Bentuk SPL dengan ruas kanan = 0, solusi trivial dan non-trivial
