HOW TO PLAY

🎯 OBJECTIVE

Decode encrypted messages using your knowledge of cipher algorithms. Each correct answer scores points — the faster you answer, the more points you earn. Build combos for multiplier bonuses!

⚡ CAESAR CIPHER

Each letter is shifted by a fixed number (the key). To decode: shift letters back by the key amount.

Example: Key = 3 → "KHOOR" decodes to "HELLO" (shift back 3)
A→X, B→Y, C→Z, D→A ... (wrap around alphabet)

🔑 VIGENÈRE CIPHER

Uses a keyword where each letter shifts by a different amount. Each letter of the key corresponds to a shift.

Key = "KEY": K=10, E=4, Y=24 — apply shifts cyclically
Decrypt: subtract key letter value from cipher letter

⊕ XOR CIPHER

XOR each ASCII value of the cipher text with the numeric key. The result is the original character code.

XOR: A⊕key = cipher, cipher⊕key = A (self-inverse!)
Shown as numbers: "72 69 76 76 79" = "HELLO" (then XOR the key)

⚡ SCORING SYSTEM

Base points per level: 100 × level. Speed bonus: up to +50% for fast answers. Combo multiplier: 2x at ×3 combo, 3x at ×6 combo, 5x at ×10 combo. Each wrong answer breaks your combo. You get 1 hint per puzzle (costs 50% of base score).
