Konsep Dasar Determinan

Definisi Formal

Determinan adalah nilai skalar yang dapat dihitung dari elemen-elemen matriks persegi dan memberikan informasi penting tentang sifat matriks tersebut.

🎯 Determinan adalah nilai skalar (bilangan tunggal) yang diperoleh dari matriks persegi (jumlah baris = kolom).

📌 Contoh: matriks 2x2, 3x3, dan seterusnya.

🧠 A. Pengertian Determinan

Determinan dinotasikan sebagai det(A) atau |A|.

💡 Digunakan untuk:

  • Menentukan apakah matriks memiliki invers
  • Menyelesaikan sistem persamaan linear

✨ det(A) ≠ 0 → Matriks memiliki invers (non-singular)

❌ det(A) = 0 → Matriks tidak memiliki invers (singular)

📐 B. Determinan Matriks 2x2

A =
ab
cd
📌 det(A) = ad - bc

🧮 Contoh:

A =
32
14
💡 det(A) = (3×4) - (2×1) = 12 - 2 = 10

📊 C. Determinan Matriks 3x3

A =
abc
def
ghi
📌 det(A) = a(ei − fh) − b(di − fg) + c(dh − eg)

⚡ Bisa dihitung dengan metode Sarrus atau Kofaktor.

🔍 D. Minor & Kofaktor

Minor (Mij) → determinan setelah menghapus baris ke-i dan kolom ke-j.

⚡ Kofaktor = (-1)i+j × Minor

📌 E. Sifat Determinan

  • ✔ Dua baris/kolom sama → determinan = 0
  • ✔ Ada baris nol → determinan = 0
  • ✔ Tukar baris → tanda berubah
  • ✔ Dikalikan k → determinan ikut dikali k
  • ✔ det(AB) = det(A) × det(B)
  • ✔ det(I) = 1

📏 F. Makna Geometris

📐 Matriks 2x2 → luas jajargenjang
📦 Matriks 3x3 → volume bangun ruang

⚙️ G. Peranan dalam SPL

Digunakan dalam Metode Cramer untuk menentukan solusi.

✅ det ≠ 0 → solusi tunggal

⚠️ det = 0 → tidak ada / banyak solusi

🎯 Kesimpulan

Determinan memiliki peranan penting dalam aljabar linear:

  • 📌 Menentukan invers matriks
  • 📌 Menyelesaikan sistem persamaan linear
  • 📌 Menjelaskan transformasi geometri
2

Teorema Penting

🎯 Determinan memiliki peranan penting dalam aljabar linear karena berkaitan dengan:

  • ✔ Sifat matriks
  • ✔ Sistem persamaan linear
  • ✔ Transformasi linear

🧠 A. Teorema Keberadaan Invers

det(A) ≠ 0 ⇔ A memiliki invers det(A) = 0 ⇔ A tidak memiliki invers

💡 Jika determinan ≠ 0 → solusi unik

⚠️ Jika determinan = 0 → tidak ada / banyak solusi

📌 Rumus: x = A⁻¹b

📊 B. Teorema Perkalian Matriks

det(AB) = det(A) × det(B)

✔ Bersifat multiplikatif

✔ Jika salah satu = 0 → hasil juga 0

💡 det(A⁻¹) = 1 / det(A)

🔁 C. Teorema Transpose

det(Aᵀ) = det(A)

📌 Baris ↔ kolom tidak mengubah nilai determinan

🔄 D. Pertukaran Dua Baris

Tukar baris → det berubah tanda

💡 det baru = - det lama

✔ Dua kali tukar → kembali seperti semula

📏 E. Perkalian Skalar

Jika satu baris dikali k → det = k × det(A)
det(kA) = kⁿ × det(A)

📌 n = ordo matriks

🚫 F. Baris / Kolom Nol

Jika ada baris nol → det(A) = 0

💡 Tidak perlu hitung → langsung 0

👯 G. Dua Baris Identik

Baris/kolom sama → det(A) = 0

📌 Karena ketergantungan linear

🧩 H. Kombinasi Linear

Jika satu baris = kombinasi baris lain → det = 0

💡 Menandakan tidak bebas linear

📐 I. Matriks Segitiga

det(A) = a₁₁ × a₂₂ × ... × aₙₙ

✔ Berlaku untuk segitiga atas & bawah

🧮 J. Ekspansi Kofaktor

det(A) = a₁₁C₁₁ + a₁₂C₁₂ + ... + a₁ₙC₁ₙ
Cij = (-1)^(i+j) × Mij

💡 Bisa dari baris atau kolom mana saja

⚙️ K. Teorema Cramer

xi = det(Ai) / det(A)

✔ det(A) ≠ 0 → solusi unik

⚠️ det(A) = 0 → tidak unik

🎯 Kesimpulan

  • ✔ det(A) menentukan invers matriks
  • ✔ det(A) = 0 → tidak invertible
  • ✔ det(AB) = det(A)det(B)
  • ✔ det(A) = det(Aᵀ)
  • ✔ Determinan berkaitan dengan luas & volume

Teknik Perhitungan

🧠 A. Metode Sarrus (3x3)

📌 Tambahkan dua kolom pertama, lalu hitung diagonal turun dan naik.

Hasil akan muncul di sini...

🔄 B. Reduksi Baris (OBE)

  • 🔁 Tukar dua baris → det = -det
  • 📏 Kali baris dengan k → det = k det
  • ➕ Tambah kelipatan baris → det tetap
💡 Gunakan metode ini untuk menyederhanakan matriks menjadi segitiga atas.

Ekspansi Kofaktor

Definisi

Ekspansi Kofaktor adalah metode untuk menghitung determinan matriks dengan mengembangkan elemen-elemen baris atau kolom tertentu.

Ekspansi kofaktor memungkinkan kita untuk menghitung determinan matriks dengan memecahnya menjadi sub-matriks yang lebih kecil. Ini sangat berguna untuk matriks berukuran besar.

class="explain"> Ekspansi kofaktor adalah metode menghitung determinan dengan mengalikan elemen dengan kofaktornya lalu menjumlahkannya.

📊 Contoh Matriks

A =
231
456
789

🔍 Minor

M₁₁ = (5 × 9) − (6 × 8)
M₁₁ =
56
89
M₁₁ = 45 − 48 = -3

⚡ Kofaktor

Cᵢⱼ = (-1)i+j × Mᵢⱼ
+−+
−+−
+−+
Contoh: C₁₁ = +(-3) = -3

🧮 Ekspansi Kofaktor

(A) = a₁₁C₁₁ + a₁₂C₁₂ + a₁₃C₁₃
(A) = (2×-3) + (3×6) + (1×-3)
= 9

🎯 Kesimpulan

Dengan menggunakan minor dan kofaktor, determinan matriks dapat dihitung secara sistematis menggunakan ekspansi kofaktor.

Aturan Cramer (Cramer's Rule)

Aturan Cramer adalah metode untuk menyelesaikan sistem persamaan linear (SPL) menggunakan determinan matriks.

📐 Rumus

xᵢ = det(Aᵢ) / det(A)
Aᵢ = matriks A dengan kolom ke-i diganti B

📊 Contoh Soal

2x + y = 5
x + 3y = 6
A =
21
13
det(A) = (2×3) - (1×1) = 5
A₁ =
51
63
det(A₁) = 9
A₂ =
25
16
det(A₂) = 7
x = 9/5 , y = 7/5

🎯 Kesimpulan

Jika det(A) ≠ 0 maka solusi bisa dicari dengan Cramer.

Sifat-Sifat Determinan

Sifat-Sifat Determinan

Beberapa sifat penting determinan:Jika dua baris/kolom sama → determinan = 0,Jika ada baris/kolom nol → determinan = 0 , Tukar dua baris → tanda determinan berubah, Dikalikan skalar k → determinan ikut dikali k, det(AB) = det(A) × det(B), det(I) = 1

✨ Sifat-sifat determinan adalah aturan-aturan yang berlaku pada determinan matriks yang membantu dalam perhitungan dan analisis matriks.

1. 🔁 Baris/Kolom Sama

Jika dua baris atau dua kolom dalam matriks sama, maka determinan = 0 ❌

2. ⛔ Baris/Kolom Nol

Jika terdapat satu baris atau kolom yang semua elemennya nol, maka determinan = 0

3. 🔄 Pertukaran Baris

Jika dua baris ditukar, maka determinan berubah tanda (±)

4. ✖️ Perkalian Skalar

Jika satu baris dikalikan dengan k, maka determinan ikut dikalikan k

5. ➕ Kombinasi Linear

Menambahkan kelipatan suatu baris ke baris lain tidak mengubah determinan ✔️

6. 🔗 Perkalian Matriks

det(AB) = det(A) × det(B)

7. 🧱 Matriks Identitas

det(I) = 1

8. 🔃 Matriks Transpose

det(Aᵀ) = det(A)

9. 📐 Matriks Segitiga

Jika matriks berbentuk segitiga, maka determinan adalah hasil kali elemen diagonal utama

10. 🧩 Ketergantungan Linear

Jika satu baris merupakan kombinasi linear dari baris lain, maka determinan = 0 ❌

🎯 Kesimpulan

Sifat-sifat determinan mempermudah perhitungan dan membantu memahami apakah suatu matriks memiliki invers serta hubungan antar baris atau kolom 📊
7

Aplikasi Determinan

🧠 Pengertian

Determinan tidak hanya digunakan dalam perhitungan matriks, tetapi juga memiliki banyak aplikasi penting dalam matematika dan bidang lainnya.

1. 📊 Menentukan Invers Matriks

Determinan digunakan untuk mengetahui apakah suatu matriks memiliki invers atau tidak.
det(A) ≠ 0 → memiliki invers
det(A) = 0 → tidak memiliki invers

2. 🧮 Menyelesaikan SPL (Cramer)

Determinan digunakan dalam Aturan Cramer untuk mencari solusi sistem persamaan linear.
xᵢ = det(Aᵢ) / det(A)

3. 📐 Menghitung Luas

Pada matriks 2×2, determinan digunakan untuk menghitung luas jajargenjang.
Luas = |det(A)|

4. 📦 Menghitung Volume

Pada matriks 3×3, determinan digunakan untuk menghitung volume bangun ruang.
Volume = |det(A)|

5. 🔄 Transformasi Linear

Determinan menunjukkan faktor perubahan skala pada transformasi linear (luas atau volume bisa membesar atau mengecil).

6. 🧩 Mengecek Ketergantungan Linear

Determinan digunakan untuk mengetahui apakah vektor saling bebas atau tidak.
det(A) = 0 → bergantung linear
det(A) ≠ 0 → bebas linear

7. 📈 Bidang Ilmu Lain

Determinan digunakan dalam berbagai bidang seperti:
  • Fisika ⚛️ (transformasi koordinat)
  • Ekonomi 💰 (model linear)
  • Teknik ⚙️ (grafika & sistem kontrol)

🎯 Kesimpulan

Determinan memiliki banyak aplikasi penting seperti menentukan invers, menyelesaikan SPL, serta menghitung luas dan volume.
