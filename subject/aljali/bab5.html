Pengantar Nilai Eigen dan Vektor Eigen

Definisi Formal

Vektor Eigen adalah vektor tak nol yang, ketika dikalikan dengan suatu matriks, hanya mengalami perubahan skala tanpa perubahan arah.

Nilai Eigen dan Vektor Eigen merupakan konsep penting dalam aljabar linear yang memiliki berbagai aplikasi dalam ilmu komputer, fisika, dan bidang lainnya. Mereka membantu dalam memahami sifat-sifat matriks dan transformasi linear.

Dalam dunia matematika, khususnya aljabar linear, nilai eigen dan vektor eigen adalah konsep penting yang digunakan untuk memahami bagaimana suatu transformasi memengaruhi suatu ruang.

✨ Konsep Dasar

Ketika sebuah matriks mengalikan suatu vektor, biasanya arah dan panjang vektor akan berubah. Namun, ada vektor tertentu yang hanya berubah panjangnya saja, tanpa berubah arah.

👉 Vektor seperti ini disebut Vektor Eigen
👉 Faktor pengalinya disebut Nilai Eigen

📐 Ilustrasi Sederhana

Bayangkan sebuah panah ➡️ yang diputar atau diregangkan. Sebagian besar panah akan berubah arah, tetapi ada panah tertentu yang tetap menunjuk ke arah yang sama — hanya panjangnya saja yang berubah 📏.

🧮 Bentuk Matematis

Jika A adalah matriks dan v adalah vektor:

Av = λv

• v = Vektor Eigen 🎯
• λ = Nilai Eigen 📊

⚙️ Kenapa Penting?

  • 📊 Analisis data (PCA)
  • 🎮 Grafika komputer
  • 📡 Sistem dinamis
  • 🤖 Machine learning
💡 Intinya:
Nilai eigen = besar skala perubahan,
Vektor eigen = arah yang tetap.

Nilai Eigen dan EigenVektor

📘 Nilai eigen adalah skalar khusus yang berhubungan dengan matriks. 📐 Vektor eigen adalah vektor tak nol yang ketika dikalikan matriks hanya berubah skala, bukan arah utamanya.
✨ Ax = λx
  • 🔢 A = matriks persegi (n × n)
  • 📍 x = vektor eigen (x ≠ 0)
  • 💡 λ = nilai eigen
↗ x
🌱 0 ≤ λ ≤ 1
↗↗ λx
🚀 λ > 1
↙ λx
🔄 -1 ≤ λ
↙↙ λx
⚡ λ ≤ -1
🎯 Secara geometris, transformasi matriks terhadap vektor eigen hanya mengubah panjang (skala) tanpa mengubah arah dasar vektor.
📌 (λI − A)x = 0
🧮 det(λI − A) = 0
🔥 Persamaan ini disebut persamaan karakteristik dan menghasilkan polinomial karakteristik untuk mencari nilai eigen.
🪜 Langkah mencari nilai eigen:
  1. ➜ Bentuk matriks λI − A
  2. ➜ Hitung determinan
  3. ➜ Selesaikan persamaan det(λI − A) = 0
  4. ➜ Dapatkan nilai λ
🧩 Langkah mencari vektor eigen:
  1. ➜ Substitusi nilai λ ke (λI − A)x = 0
  2. ➜ Bentuk sistem persamaan linear
  3. ➜ Cari solusi vektor x
📊 Sifat-sifat penting nilai eigen:
  • ➕ Jumlah semua nilai eigen = trace matriks
  • ✖️ Hasil kali semua nilai eigen = determinan matriks
  • ⚠️ Jika det(A) = 0 → minimal satu λ = 0
  • 📐 Matriks diagonal → λ = elemen diagonal
  • 🔁 Matriks simetris → nilai eigen real
🚀 Aplikasi nilai eigen:
  • 📐 Transformasi linear (rotasi, scaling)
  • 📊 Analisis data (PCA di Machine Learning)
  • ⚙️ Sistem dinamik & stabilitas
  • 🌐 Analisis jaringan & graf
💡 Nilai eigen dan vektor eigen digunakan untuk memahami bagaimana suatu matriks mengubah bentuk, arah, dan skala suatu ruang.

Persamaan Vektor Eigen

🎯 Pengertian Nilai & Vektor Eigen

Nilai eigen adalah angka khusus yang menunjukkan seberapa besar perubahan skala pada suatu vektor ketika dikalikan dengan matriks.

A x = λ x
  • A = matriks
  • x = vektor eigen
  • λ = nilai eigen
✨ Vektor eigen tidak berubah arah, hanya berubah panjang

🧮 Cara Mencari Nilai Eigen

det(λI − A) = 0
  1. 🔍 Bentuk matriks (λI − A)
  2. 🧠 Hitung determinan
  3. 📊 Bentuk persamaan polinomial
  4. ✨ Cari akar (nilai λ)

📌 Cara Mencari Vektor Eigen

(λI − A)x = 0
  1. Pilih salah satu nilai λ
  2. Substitusi ke persamaan
  3. Selesaikan sistem linear
  4. Dapatkan vektor eigen
✨ λ₁ = 2, λ₂ = 3

Matriks diagonal → nilai eigen langsung dari diagonal utama.

📊 Jenis-Jenis Nilai Eigen

  • 🔢 Real → bilangan biasa
  • 🌀 Kompleks → mengandung i
  • 🔁 Berulang → nilai sama muncul lebih dari sekali
  • 0️⃣ Nol → menunjukkan matriks tidak invertible

🧩 Diagonalisasi Matriks

A = P × D × P⁻¹

Matriks bisa didiagonalkan jika memiliki cukup vektor eigen.

🚀 Mempermudah perhitungan besar

⚡ Pangkat Matriks Cepat

Aⁿ = P × Dⁿ × P⁻¹
🔥 Digunakan untuk menghitung A⁵⁰, A¹⁰⁰, dll

📊 Hubungan dengan Determinan

det(A) = λ₁ × λ₂ × ... × λₙ
❗ Jika ada λ = 0 → matriks tidak punya invers

📌 Trace Matriks

tr(A) = λ₁ + λ₂ + ... + λₙ
✨ Jumlah nilai eigen = trace

📉 Stabilitas Sistem

  • ✅ Semua λ negatif → stabil
  • ⚠️ Ada λ positif → tidak stabil
  • 🔁 λ = 0 → kritis

📐 Makna Geometris

➡️ arah tetap
📏 membesar
📉 mengecil
🔄 berbalik
🎯 Vektor eigen hanya berubah panjang

🌍 Aplikasi Nilai Eigen

  • 🤖 Machine Learning (PCA)
  • 📊 Analisis data
  • 🔊 Pengolahan sinyal
  • 🏗️ Teknik struktur
  • 💻 Grafika komputer
🚀 Dipakai di AI, teknik, dan sains

⭐ Kenapa Penting?

  • Mempermudah perhitungan matriks
  • Menentukan sifat sistem
  • Digunakan di banyak bidang

🎯 Kesimpulan

Nilai eigen dan vektor eigen adalah konsep penting dalam aljabar linear yang digunakan untuk memahami transformasi, stabilitas, dan berbagai aplikasi modern.

Matriks dan Transformasi Linear

🎯 Matriks & Transformasi Linear

Matriks adalah susunan bilangan dalam baris dan kolom yang digunakan untuk merepresentasikan transformasi linear dalam matematika.

📊 Bentuk Matriks

a₁₁   a₁₂   a₁₃
a₂₁   a₂₂   a₂₃
a₃₁   a₃₂   a₃₃

Matriks memiliki ukuran m × n (baris × kolom).

➕ Operasi Matriks

  • Penjumlahan matriks
  • Perkalian dengan skalar
  • Perkalian matriks
C = A × B

🔄 Transformasi Linear

Transformasi linear adalah fungsi yang memetakan vektor ke vektor lain dengan mempertahankan operasi penjumlahan dan perkalian skalar.

T(x) = A × x

📌 Representasi Vektor

x₁
x₂

Vektor direpresentasikan sebagai matriks kolom.

🧠 Contoh Transformasi

Misalkan matriks:

2   0
0   3

Matriks ini memperbesar sumbu x sebanyak 2 kali dan sumbu y sebanyak 3 kali.

📐 Jenis Transformasi Linear

  • 🔍 Dilatasi (perbesaran)
  • 🔄 Rotasi (perputaran)
  • ↔️ Refleksi (pencerminan)
  • 📏 Shear (geser)

🔄 Matriks Rotasi

cosθ   -sinθ
sinθ   cosθ
🎯 Digunakan untuk memutar vektor

🪞 Matriks Refleksi

Terhadap sumbu x:

1   0
0   -1

📏 Matriks Shear

1   k
0   1

📊 Determinan dalam Transformasi

det(A)
  • 📐 Menentukan perubahan luas/volume
  • ❗ det = 0 → ruang runtuh
  • 🔁 det negatif → orientasi terbalik

⭐ Hubungan dengan Eigen

Nilai eigen menunjukkan faktor skala, sedangkan vektor eigen menunjukkan arah tetap dalam transformasi.

A x = λ x

🌍 Aplikasi Transformasi Linear

  • 🎮 Grafika komputer
  • 🤖 Machine Learning
  • 📊 Analisis data
  • 🏗️ Teknik sipil

🎯 Kesimpulan

Matriks adalah alat utama dalam transformasi linear. Dengan matriks, kita dapat mengubah bentuk, arah, dan ukuran suatu objek secara matematis.

Persamaan Karakteristik

💡 Dalam aljabar linear, nilai eigen dan vektor eigen digunakan untuk memahami bagaimana suatu matriks mentransformasikan suatu vektor.
Ax = λx
📌 Artinya: vektor x tidak berubah arah, hanya berubah skala sebesar λ.

🔍 A. Pengertian Nilai Eigen & Vektor Eigen

  • A = matriks persegi
  • x = vektor eigen (vektor tak nol)
  • λ = nilai eigen (skalar)
✨ Vektor eigen adalah vektor yang ketika dikalikan matriks hanya berubah panjang, bukan arah.

⚙️ B. Persamaan Karakteristik

Untuk mencari nilai eigen, digunakan persamaan karakteristik.
det(λI − A) = 0
📌 Persamaan ini menghasilkan polinomial karakteristik.

📐 C. Matriks Identitas (I)

I =
1 0
0 1
📌 Matriks identitas digunakan dalam persamaan λI − A.

🧮 D. Langkah Mencari Nilai Eigen

  1. 🔹 Bentuk persamaan: det(λI − A)
  2. 🔹 Hitung determinan
  3. 🔹 Bentuk persamaan polinomial
  4. 🔹 Cari akar → nilai eigen λ

📊 E. Contoh Soal (2x2)

Diketahui matriks:
12
34
Langkah 1: Bentuk λI − A
λ−1-2
-3λ−4
Langkah 2: Hitung determinan
(λ−1)(λ−4) − (-2)(-3) = 0
λ² − 5λ − 2 = 0
🎯 Nilai eigen diperoleh dari akar persamaan tersebut.

📌 F. Mencari Vektor Eigen

Setelah mendapatkan λ, substitusi ke:
(λI − A)x = 0
📌 Hasilnya adalah sistem persamaan linear untuk mencari vektor x.

📦 G. Contoh Vektor Eigen

Misalkan λ = 2, maka:
12
-1-2
Dari sistem:
v₁ + 2v₂ = 0
👉 v₁ = -2v₂
✨ Jadi vektor eigen:
x = ( -2 , 1 )

📈 H. Makna Geometris

  • 📏 λ > 1 → vektor membesar
  • 📉 0 < λ < 1 → mengecil
  • 🔄 λ < 0 → arah berlawanan
  • ⚠️ λ = 0 → runtuh ke titik nol

⚡ I. Sifat-Sifat Nilai Eigen

  • Jumlah nilai eigen = trace matriks
  • Hasil kali nilai eigen = determinan
  • Matriks segitiga → eigen = elemen diagonal

🎯 J. Aplikasi

  • 📊 Machine Learning
  • 📈 Analisis data (PCA)
  • 🎮 Grafik komputer
  • 🔬 Fisika & teknik

🎯 Kesimpulan

✨ Persamaan karakteristik digunakan untuk mencari nilai eigen.
✨ Nilai eigen menunjukkan faktor skala transformasi.
✨ Vektor eigen menunjukkan arah yang tetap setelah transformasi.

Diagonalization Matriks

💡 Diagonalization (Diagonalisasi) adalah proses mengubah suatu matriks menjadi bentuk matriks diagonal menggunakan transformasi tertentu.
✨ Matriks diagonal adalah matriks yang hanya memiliki elemen tidak nol pada diagonal utama.

🔷 A. Bentuk Matriks Diagonal

λ₁00
0λ₂0
00λ₃
📌 Elemen diagonal biasanya adalah nilai eigen.

⚙️ B. Konsep Diagonalisasi

A = PDP⁻¹
📌 Di mana:
  • A = matriks asal
  • P = matriks vektor eigen
  • D = matriks diagonal (isi nilai eigen)
  • P⁻¹ = invers dari P

🔍 C. Syarat Matriks Bisa Didagonal

Suatu matriks dapat didiagonalkan jika:
  • ✅ Memiliki cukup vektor eigen (bebas linear)
  • ✅ Jumlah vektor eigen = ukuran matriks
  • ✅ Matriks dapat dibentuk menjadi P yang invertible
⚠️ Jika tidak memenuhi, maka matriks tidak bisa didiagonalkan.

🧮 D. Langkah-Langkah Diagonalisasi

  1. 🔹 Cari nilai eigen (λ)
  2. 🔹 Cari vektor eigen untuk setiap λ
  3. 🔹 Susun matriks P dari vektor eigen
  4. 🔹 Susun matriks D dari nilai eigen
  5. 🔹 Hitung P⁻¹
  6. 🔹 Gunakan rumus A = PDP⁻¹

📊 E. Contoh Soal

Diketahui matriks:
21
03

1️⃣ Nilai Eigen

(2 − λ)(3 − λ) = 0
🎯 λ₁ = 2 dan λ₂ = 3

2️⃣ Vektor Eigen

Untuk λ₁ = 2:
01
01
👉 vektor eigen: (1,0)
Untuk λ₂ = 3:
👉 vektor eigen: (1,1)

3️⃣ Matriks P

11
01

4️⃣ Matriks D

20
03

📐 F. Manfaat Diagonalisasi

  • ⚡ Mempermudah perhitungan pangkat matriks
  • 📊 Digunakan dalam analisis sistem dinamis
  • 🧠 Penting dalam machine learning (PCA)
  • 📈 Digunakan dalam grafik komputer

📦 G. Pangkat Matriks Lebih Mudah

Aⁿ = PDⁿP⁻¹
✨ Karena D diagonal, maka Dⁿ sangat mudah dihitung.

📉 H. Jika Tidak Bisa Diagonal

⚠️ Jika matriks tidak memiliki cukup vektor eigen:
  • Tidak bisa didiagonalkan
  • Digunakan bentuk lain (Jordan Form)

🎯 Kesimpulan

✨ Diagonalisasi mengubah matriks menjadi bentuk sederhana.
✨ Menggunakan nilai eigen dan vektor eigen.
✨ Mempermudah banyak perhitungan matematika lanjutan.

Aplikasi Nilai Eigen dan Vektor Eigen

📊 Aplikasi Nilai Eigen & Vektor Eigen

💡 Nilai eigen (λ) dan vektor eigen (x) tidak hanya konsep teori, tetapi memiliki banyak aplikasi penting dalam berbagai bidang ilmu.
Ax = λx
📌 Persamaan ini menunjukkan bagaimana suatu transformasi hanya mengubah skala vektor.

🧠 A. Aplikasi dalam Machine Learning

✨ Dalam Machine Learning, nilai eigen digunakan dalam teknik PCA (Principal Component Analysis).
  • 📊 Mengurangi dimensi data
  • 📉 Menghilangkan noise
  • 📈 Menemukan fitur paling penting
👉 Vektor eigen menunjukkan arah variasi terbesar data.

📊 B. Analisis Data & Statistik

Dalam statistik, nilai eigen digunakan pada matriks kovarians:
σ₁₁σ₁₂
σ₂₁σ₂₂
  • 📌 Menentukan arah distribusi data
  • 📌 Mengukur variasi data

🎮 C. Grafik Komputer

Dalam grafik komputer, digunakan untuk:
  • 🎨 Transformasi objek (rotasi, scaling)
  • 🧊 Animasi 3D
  • 📐 Rendering gambar
✨ Vektor eigen menjaga arah objek tetap stabil saat transformasi.

⚙️ D. Sistem Dinamis

Digunakan untuk menganalisis kestabilan sistem:
xₙ₊₁ = Axₙ
  • 📈 λ > 1 → sistem tidak stabil
  • 📉 0 < λ < 1 → sistem stabil
  • 🔄 λ negatif → osilasi

🔌 E. Teknik & Fisika

Digunakan dalam:
  • ⚡ Analisis getaran
  • 🏗️ Struktur bangunan
  • 🔊 Gelombang & frekuensi alami
📌 Nilai eigen = frekuensi alami sistem

🌐 F. Google Search (PageRank)

Algoritma PageRank menggunakan nilai eigen untuk menentukan:
  • 🌍 Ranking halaman web
  • 🔗 Kepentingan suatu halaman
Ax = x
📌 Ini adalah kasus khusus λ = 1

🧬 G. Biologi & Populasi

Digunakan dalam model pertumbuhan populasi:
xₙ = Aⁿx₀
  • 📈 λ terbesar → laju pertumbuhan
  • 📉 λ kecil → penurunan populasi

📦 H. Kompresi Gambar

Digunakan dalam:
  • 🖼️ Kompresi gambar
  • 📉 Mengurangi ukuran file
  • ✨ Menjaga kualitas penting

🎯 I. Makna Geometris

  • 📏 λ → faktor skala
  • 📐 vektor eigen → arah tetap
  • 🔄 transformasi → tidak mengubah arah eigen

🎯 Kesimpulan

✨ Nilai eigen & vektor eigen sangat penting dalam banyak bidang.
✨ Digunakan dalam AI, teknik, grafik, dan sains.
✨ Membantu memahami transformasi dan sistem kompleks.
