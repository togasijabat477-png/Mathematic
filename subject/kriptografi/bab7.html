II. Modular Arithmetic Operations

A. Commutative Laws

  • 1. (w + x) mod n = (x + w) mod n
  • 2. (w × x) mod n = (x × w) mod n

Contoh:

1. w=6, x=4, n=3 → (6+4) mod 3 = 10 mod 3 = 1
2. w=2, x=5, n=4 → (2×5) mod 4 = 10 mod 4 = 2

B. Modular Arithmetic Rules

  1. [(a mod n) + (b mod n)] mod n = (a + b) mod n
  2. [(a mod n) - (b mod n)] mod n = (a - b) mod n
  3. [(a mod n) × (b mod n)] mod n = (a × b) mod n
Contoh Pengurangan: a=37, b=3, n=10 [(37 mod 10) - (3 mod 10)] mod 10 = (37 - 3) mod 10 [7 - 3] mod 10 = 34 mod 10 4 = 4

C. Distributive Law

[w × (x + y)] mod n = [(w × x) + (w × y)] mod n
Contoh: w=20, x=21, y=22, n=23 [20 × (21 + 22)] mod 23 = [(20 × 21) + (20 × 22)] mod 23 860 mod 23 = 860 mod 23 9 = 9

D. Additive Inverse (-w)

Untuk setiap w ∈ Zn, terdapat z sedemikian hingga w + z ≡ 0 mod n.

Contoh: w = 42, n = 9 42 + z ≡ 0 (mod 9) z ≡ -42 (mod 9) -42 ≡ -42 + (9 × 5) = 3 Maka inversnya adalah 3 karena 42 + 3 = 45 ≡ 0 (mod 9)

E. Property: Identities

1. (0 + w) mod n = w mod n

2. (1 * w) mod n = w mod n

F. Eksponensial Modulo

Expression: xa (mod n)
Contoh 1: 2^11 mod 13 2^2 = 4 2^4 = 16 ≡ 3 mod 13 2^8 = 3^2 = 9 mod 13 2^11 = 2^8 * 2^2 * 2^1 = 9 * 4 * 2 = 72 72 mod 13 = 7 (Karena 13 * 5 = 65, sisa 7) Contoh 2: 52^11 mod 13 52 = 13 * 4 ≡ 0 mod 13 Maka 52^11 ≡ 0^11 ≡ 0 mod 13

III. Caesar Cipher

"Dapat direpresentasikan dengan menyelaraskan plaintext dengan ciphertext ke kiri atau kanan sebanyak jumlah pergeseran."

Rumus Matematis:
Enkripsi: \(C = (P + K) \pmod{26}\)
Dekripsi: \(P = (C - K) \pmod{26}\)

IV. Playfair Cipher

Dikembangkan oleh Charles Wheatstone (1854). Menggunakan matriks 5x5 dengan ketentuan huruf I = J.

Aturan Enkripsi:

  • Jika baris sama: geser ke kanan.
  • Jika kolom sama: geser ke bawah.
  • Jika berbeda baris & kolom: membentuk persegi panjang (ambil sudut lawan).
