Aljabar Boolean – Matematika Diskrit

1. Pengertian Aljabar Boolean

Aljabar Boolean adalah sistem matematika yang bekerja hanya menggunakan dua nilai logika yaitu 0 (False) dan 1 (True). Dikembangkan oleh George Boole, konsep ini menjadi dasar dari semua sistem digital dan komputer modern.

Aljabar Boolean digunakan dalam:

  • Perancangan rangkaian digital
  • Pemrograman logika (struktur IF, AND, OR)
  • Sistem komputer dan prosesor
  • Sistem kecerdasan buatan

2. Nilai Logika dan Variabel Boolean

Dalam Boolean, setiap variabel hanya memiliki dua kemungkinan nilai:

NilaiMakna
1Benar / True
0Salah / False

Variabel Boolean dilambangkan dengan huruf kapital seperti A, B, C.

4. Operator Turunan

NAND (NOT AND)

Merupakan kebalikan dari operasi AND.

NOR (NOT OR)

Merupakan kebalikan dari OR.

XOR (Exclusive OR)

Hasilnya benar jika kedua nilai berbeda.

5. Hukum-Hukum Dasar Aljabar Boolean

  • Identitas: A + 0 = A, A · 1 = A
  • Dominasi: A + 1 = 1, A · 0 = 0
  • Komutatif: A + B = B + A
  • Asosiatif: (A + B) + C = A + (B + C)
  • Distributif: A · (B + C) = A·B + A·C
  • Komplemen: A + A' = 1, A · A' = 0

6. Hukum De Morgan

(A + B)' = A' · B'

(A · B)' = A' + B'
